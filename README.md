# SolidSnake 🐍

**Snake game written in SolidJS for fun and no profit.**

Still WIP with bugs, too bad the snake is a vegan.

![image description](src/assets/snake.gif)

## Controls

- **Space** to start
- **W**, **A**, **S**, **D**

## Rules

- Collect tomatoes
- Don't hit the wall
- Don't hit yourself
- Have fun!
